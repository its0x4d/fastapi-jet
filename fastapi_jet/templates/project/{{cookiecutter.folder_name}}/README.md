# {{ cookiecutter.name }}

A modular FastAPI project generated by [FastAPI-Jet](https://github.com/its0x4d/fastapi-jet).

## Quick start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
fastjet runserver
```

Open http://127.0.0.1:8000/docs for the API explorer.

## Add a new app

```bash
fastjet startapp users
fastjet startapp billing --crud
```

Then register it in `INSTALLED_APPS` inside `base/main.py` (the CLI prints the line):

```python
INSTALLED_APPS: list[AppRoute] = [
    AppRoute(name="core", prefix="/core", tags=["core"]),
    AppRoute(name="users", prefix="/users", tags=["users"]),
]
```

## Useful commands

```bash
fastjet runserver      # Start dev server
fastjet apps           # List installed apps
fastjet routes         # List all routes
fastjet check          # Validate project structure
fastjet shell          # Python shell with app loaded
pytest                 # Run tests
```

See [FastAPI-Jet conventions](https://github.com/its0x4d/fastapi-jet/blob/main/docs/CONVENTIONS.md) for app structure (`routers`, `schemas`, `services`, `dependencies`).

## Configuration

Copy `.env.example` to `.env` and adjust values. Settings are loaded in `base/core.py`.
